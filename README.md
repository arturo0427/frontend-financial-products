# FrontProducts

Aplicacion Angular para gestionar productos financieros.

## Requisitos

- Node.js (recomendado LTS)
- npm (incluido con Node)
- Angular CLI 19.x

## Instalacion

```bash
npm install
```

## Ejecutar en desarrollo (con proxy)

Por CORS del backend se usa proxy.

```bash
ng serve --proxy-config proxy.conf.json
```

Abre `http://localhost:4200/`.

## Ejecutar pruebas unitarias

```bash
ng test
```

## Build

```bash
ng build
```
